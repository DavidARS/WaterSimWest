Groundwater Management

The policy choice for Groundwater changes the amount of groundwater available to be pumped for each Consumer of Water that has access to groundwater resources. The input buttons for this control range from "Least" (minimum amount of available groundwater) to "Most" (maximum amount of groundwater available). To the right you will notice an input box that enables the user to enter a desired value to use.

Decrease the available Groundwater to safeguard use for another time. Increase the available Groundwater if you find that no other options suit your needs.

NOTE:If the background color to the user input text box changes color, the groundwater water value has been adjusted by model to the value displayed: estimated demand is less than the available water supply.

In addition: 1) the model will limit the lower and upper boundaries of viable input values for this policy, and 2) the model will NOT let you allocate more water than the estimated demand for water. It will adjust water supplies as needed.

Field name: GWM_P
